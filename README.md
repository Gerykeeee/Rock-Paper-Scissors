# Rock-Paper-Scissors
Ez egy objektumorientált (OOP) Kő-Papír-Olló játék Pythonban. A hagyományos, átláthatatlan if-else láncok helyett professzionális megoldásokat használ: Enumokat a szigorú típusbiztonságért, és szótár-alapú (Hash Map) logikát, amivel a kiértékelés villámgyors, O(1) időkomplexitású. Tiszta, skálázható és hibatűrő kód.
